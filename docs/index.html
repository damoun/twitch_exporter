Twitch Exporter Token Helper

Generate access tokens for the Twitch Exporter

This tool helps you obtain a Twitch access token with the required scopes for the exporter. You need to create a Twitch application first.

Add this URL to your Twitch app's OAuth Redirect URLs

Subscriber Metrics
channel:read:subscriptions Read subscriber count
EventSub (Chat Messages)
channel:bot Join channel as bot
user:read:chat Read chat messages
user:bot Act as bot user

Success

Security Note: Never share your tokens. This page runs entirely in your browser - no data is sent to any server.
